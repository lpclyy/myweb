# 企业官方网站

一个功能完整的企业官方网站，包含企业信息展示、项目展示、多角色登录系统和权限管理功能。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000 查看网站

### 构建生产版本

```bash
npm run build
```

## ✨ 功能特性

### 1. 企业信息展示
- **首页**：英雄区域、核心价值观、精选项目展示
- **关于我们**：企业使命、发展历程时间轴、管理团队介绍
- **联系我们**：联系方式展示、在线留言表单

### 2. 项目展示系统
- 项目列表页面
- 按类别和状态筛选
- 项目详情页面
- 项目图片展示

### 3. 多角色登录系统
支持三种独立的登录模式：
- **顾客登录**：访问公开的企业信息和项目展示
- **员工登录**：查看内部信息和处理工作任务
- **管理员登录**：完整的系统管理权限

### 4. 权限管理体系
- **管理员权限**：
  - 企业信息管理
  - 用户管理
  - 项目管理
  - 系统配置
- **员工权限**：
  - 工作台访问
  - 内部信息查看
- **顾客权限**：
  - 公开内容访问

### 5. UI/UX 设计
- 简约大气的设计风格
- 专业的蓝色主题配色
- 清晰的信息层级
- 完全响应式设计
- 流畅的动画效果

## 🛠️ 技术栈

- **前端框架**：React 18
- **开发语言**：TypeScript
- **构建工具**：Vite
- **路由管理**：React Router v6
- **状态管理**：React Context API
- **样式**：CSS3 (模块化样式)
